## What is Rubik's Vintage?

Rubik's Vintage is a full-stack e-commerce platform built specifically for selling vintage jewelry online. It combines a beautiful customer-facing storefront with powerful admin tools to manage inventory, orders, and customer relationships.

The platform is built on modern web technologies including Next.js 16, React 19, Supabase for authentication and data storage, and Stripe for secure payment processing.

## Built With

Rubik's Vintage leverages industry-standard technologies:

- **Next.js 16** - App Router for server-side rendering and routing
- **React 19** - Modern UI with server components
- **Supabase** - PostgreSQL database with real-time capabilities and authentication
- **Stripe** - PCI-compliant payment processing
- **Tailwind CSS 4** - Responsive design system
- **TypeScript** - Type-safe development
